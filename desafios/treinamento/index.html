Vamos resumir algumas coisa

  1. Vamos começar sabendo que o (h1) é o assunto principal.
  2. O (h2) é o sub-assunto
  3. O (h3) há é o sub-assunto do (h2)

Algumas tags

  1. A Tag (EM) é para destacar a mensagem.
  2. A Tag (STRONG) é para usar em negrito.
  3. A Tag (STYLE) é um estilo, style é dp CSS
  4. A Tag (MARK) é um marcador de texto.
  5. A Tag (SMALL) pode deixar o texto pequeno
  6. A Tag (DELL) para indicar que o texto deve ser lido, mas não considerado.
  7. A Tag (INS) pode dar umaênfase e indicar que ele foi adicionado depois. (tem sentido/semantica)
  8. A Tag (SUP) para sobrescrito. EX:x20+3
  9. A Tag (SUB) para texto subscrito. EX: H2O
  10. A Tag (CODE) é uma fonte mono-espaçada
    .
  11. A Tag (PRÉ) serve para arrumar a Tag (CODE). Como esta ali em cima.
  12. A Tag (Q) é aspas. É bom usar essa tag porque é um sentido de formação (semantica).Usar quando ouver uma citação.
  13. A Tag (BLOCKQUOTE) é uma citação também,mas ela é mais longa do que a outra, que foi a Tag (Q).
  14. A Tag (CITE) ela cita URLs, pode colocar essa Tag dentro da Tag (Blockquote).
  15. A Tag (BDO) ela serve para inverter a frase.
  16. A Tag (OL) é listas ordenadas. E dentro dessa Tag, colocamos a Tag (li), para numerar as frases de forma certa.
  17. A Tag (TYPER="") se coloca dentro da Tag (OL). Essa Tag serve para mudar a numeração ou as letras. EX: 1.ACORDAR. 2.ESCOVAR OS DENTES. 3.TOMAR CAFÉ. O (TYPER) pode mudar os números para: A,B,C.
